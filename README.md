# Sales Performance Dashboard

## Overview
An end-to-end business intelligence project simulating a real 
world analyst workflow. Built using SQL and Power BI to analyze 
sales and profitability performance for a fictional US retail 
company.

## Business Problem
Regional managers need visibility into sales and profitability 
performance across regions, product categories, and customer 
segments to identify gaps and make informed decisions.

## Tools Used
- SQL (data cleaning & transformation)
- Power BI (dashboard & visualization)
- GitHub (documentation & version control)

## Dataset
- **Source:** Kaggle - Superstore Dataset
- **Link:** https://www.kaggle.com/datasets/vivek468/superstore-dataset-final
